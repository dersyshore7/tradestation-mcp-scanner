# tradestation-mcp-scanner

Starter project for an MCP-based options scanner.

## Project structure

```text
src/
  app/
    runScan.ts
  mcp/
    server.ts
  openai/
    client.ts
  scanner/
    scoring.ts
  tradestation/
    client.ts
  index.ts
```

## Run locally

```bash
npm install
npm run dev
```

## Build and run compiled output

```bash
npm run build
npm run start
```
